# Future_DS_01 Superstore Sales Performance Analytics

## Project Overview
An interactive sales and profit performance dashboard built using **Tableau Public** for the Future Interns data science for the future interns internship task. It tracks high-level business metrics, regional trends, product velocity, and how profit margins are.

## Dashboard Preview
![Dashboard Screenshot](Screenshot%20(2616).jpg)

## Key Features
* **KPI Blocks:** Quick view of total Sales, Profits, and Orders.
* **Profit by Sub-Category:** Identifies exactly which items drive margin growth.
* **Sales by Region:** Displays revenue performance across major territories.
* **Top 10 Products:** Spotlights the highest-selling SKU items.
* **Sales Trend:** Multi-year timeline tracking consumer demand.

## Tools Used
* Tableau Public
* Sample Superstore Dataset (Orders Table)
